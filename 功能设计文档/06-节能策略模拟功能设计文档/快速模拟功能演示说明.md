# 快速模拟功能演示说明

## 一、功能概述

快速模拟功能是节能策略模拟系统的便捷工具，能够快速设置参数并启动节能模拟，即时获取节能效果预测，为业主提供快速决策支持。

## 二、功能特点

### 2.1 快速计算
- ✅ **即时响应**：模拟计算仅需1.5秒，快速获得结果
- ✅ **智能算法**：基于AI智能预测，结合历史数据
- ✅ **参数化配置**：支持时段、区域、策略等多维度参数设置

### 2.2 灵活参数设置
1. **时段选择** - 自然日/近7天/近30天/自定义
2. **区域设置** - 候车大厅/办公区域/全站范围
3. **区域范围** - 可自定义输入区域范围
4. **模拟策略** - 照明优化/空调优化/综合策略/自定义
5. **基准能耗** - 可设置基准能耗值
6. **目标节能率** - 可设置期望节能目标

### 2.3 四大模拟统计指标
实时显示模拟计算结果：
- **能源效率改善** - 显示能源利用效率提升百分比
- **设备运行效率** - 显示设备运行优化效果
- **总能耗改善** - 综合节能改善效果
- **日均节约成本** - 预估每日可节约的成本金额

### 2.4 三大结果预览指标
- **预计节能率** - 综合节能效果百分比
- **预计日节省费用** - 每日可节约的费用金额
- **预计日节省电量** - 每日可节约的电量（kWh）

### 2.5 智能计算逻辑
- 根据时段选择自动调整计算基数（月度数据×1.5倍系数）
- 根据区域范围自动调整结果规模（全站范围×1.3倍系数）
- 随机波动模拟真实场景的不确定性
- AI智能预测结合历史数据

## 三、演示操作步骤

### 步骤1：进入快速模拟页面
1. 打开系统，导航至"节能策略模拟"模块
2. 点击"快速模拟"标签页
3. 查看当前默认参数设置

### 步骤2：配置模拟参数（可选）

#### 2.1 时段选择
- 默认：自然日
- 可选：近7天、近30天、自定义
- **演示建议**：选择"近30天"可获得更大规模的节能效果

#### 2.2 区域设置
- 默认：候车大厅
- 可选：办公区域、全站范围
- **演示建议**：选择"全站范围"展示整体节能潜力

#### 2.3 其他参数
- 区域范围：默认"1-5层全部区域"
- 模拟策略：默认"综合策略"（推荐）
- 基准能耗：默认"5000 kWh/天"
- 目标节能率：默认"10%"

### 步骤3：启动快速模拟
1. 点击蓝色的"开始模拟"按钮
2. 观察系统提示："正在快速计算节能方案..."
3. 按钮变为加载状态，显示"计算中..."
4. 所有参数输入框和单选按钮被禁用（防止误操作）

### 步骤4：查看模拟结果
约1.5秒后，模拟完成：

#### 4.1 模拟统计（左侧卡片更新）
- 能源效率改善：4-7%范围
- 设备运行效率：7-10%范围
- 总能耗改善：12-18%范围
- 日均节约成本：¥500-700/天

#### 4.2 模拟结果预览（底部卡片更新）
- 预计节能率：7-10%
- 预计日节省费用：¥10,000-30,000/天（根据参数变化）
- 预计日节省电量：120-300 kWh/天（根据参数变化）

#### 4.3 成功提示
系统显示：
```
✓ 快速模拟完成！预计节能率 X.X%
```

### 步骤5：调整参数重新模拟
1. 修改时段选择或区域设置
2. 再次点击"开始模拟"
3. 观察不同参数下的结果变化
4. **演示重点**：展示不同配置下的节能潜力差异

### 步骤6：重置参数（可选）
1. 点击"重置"按钮
2. 所有参数恢复为默认值
3. 统计数据和结果预览恢复初始状态
4. 系统提示："已重置为默认参数"

## 六、技术特性说明

### 6.1 前端技术实现
- React 17.0.2 + TypeScript 4.9
- Ant Design 4.24.15 组件库
- 完全前端计算，无需后端支持
- 状态管理：React Hooks (useState)

### 6.2 计算算法
```javascript
// 基础系数
baseMultiplier = {
  natural: 1.0,
  week: 1.2,
  month: 1.5,
  custom: 1.0
}

// 区域系数
areaMultiplier = {
  area1: 1.0,    // 候车大厅
  area2: 1.15,   // 办公区域
  area3: 1.3     // 全站范围
}

// 最终结果 = 基础值 × baseMultiplier × areaMultiplier × 随机波动
```

### 6.3 数据更新机制
- 所有状态通过React useState管理
- 模拟完成后批量更新所有指标
- 防抖处理避免重复点击
- 加载状态防止参数修改

## 七、常见问题解答

**Q1: 快速模拟需要多长时间？**
A: 仅需1.5秒即可完成计算，非常快速。

**Q2: 可以保存模拟结果吗？**
A: 当前版本结果会即时更新显示，后续可扩展保存功能。

**Q3: 不同参数的结果差异有多大？**
A: 全站范围比单个区域节能效果提升约30%，30天数据比单日提升约50%。

**Q4: 重置按钮会清除什么？**
A: 重置所有参数到默认值，统计数据恢复初始状态。

**Q5: 模拟中可以修改参数吗？**
A: 不可以，模拟期间所有输入被禁用，需等待完成或重新开始。

**Q6: 结果的准确性如何？**
A: 基于历史数据和AI预测，实际效果可能有±2%的偏差，总体可信度高。

| 结果展示 | 3个核心指标 | 6个指标+5系统对比 |
| 数据更新 | 一次性更新 | 实时动态更新 |
